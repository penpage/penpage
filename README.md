# PenPage

**AI speaks Markdown. You got PenPage.**

Write beautifully. Stay private. Go anywhere.

[**Try PenPage — penpage.com**](https://penpage.com)

---

Every AI agent — Claude, ChatGPT, Cursor — reads and writes Markdown. CLAUDE.md, rules files, prompts, plans — they're all `.md` files.

**PenPage** lets you create, edit, and manage them beautifully — no syntax to memorize, no messy plain-text editors.

---

## Why PenPage

**You type. It looks perfect.** Real-time WYSIWYG that outputs clean Markdown — what you see is what AI reads.

**Your data never leaves your hands.** Notes live on YOUR device. Optionally sync to YOUR Google Drive. We never touch your data.

**Two-way link to CLAUDE.md.** Link any local folder to PenPage. Edit `CLAUDE.md` in a beautiful editor — when AI updates the file, PenPage syncs automatically. Edit in PenPage, the local file updates too. Bidirectional. Works with any `.md` file on your machine.
